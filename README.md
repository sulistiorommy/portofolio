## 🏁 Getting Started

Follow these steps to set up the project locally:

### 1. Clone the Repository
```bash
git clone https://github.com/sulistiorommy/portofolio.git
cd portofolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Set Up Environment Variables
Create a `.env` file in the root directory and add your credentials. You can use `.env.example` as a template:
```bash
cp .env.example .env
```

**Required variables:**
- `VITE_GITHUB_TOKEN`: Your GitHub Personal Access Token.
- `VITE_WAKATIME_API_KEY`: Your WakaTime API Key.
- `VITE_FORMSPREE_ID`: Your Formspree form ID for the contact section.
- `VITE_UMAMI_WEBSITE_ID`: Your Umami Website ID.
- `VITE_UMAMI_SHARE_TOKEN`: Your Umami Share Token (for dashboard stats).
- `VITE_UMAMI_SERVER_URL`: Your Umami instance URL (default: `https://cloud.umami.is`).


### 4. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.
